This repository contains cybersecurity projects developed as part of the CodeAlpha Internship.
The projects focus on understanding network traffic, packet analysis, and intrusion detection through hands-on implementation.

🕵️‍♂️ Project 1: Basic Network Sniffer
🔍 What this project does

The Basic Network Sniffer captures live network packets and displays important packet details such as:

Source IP address

Destination IP address

Protocol used

It helps in understanding how data flows through a network and how packets are transmitted between devices.

⚙️ How it works

Listens to network traffic in real time

Captures packets using Python libraries

Extracts and displays packet information for analysis

🎯 Purpose

Learn packet sniffing concepts

Understand network communication

Build a foundation in cybersecurity and networking

🚨 Project 2: Network Intrusion Detection System (NIDS)
🔍 What this project does

The Network Intrusion Detection System (NIDS) monitors network traffic and detects suspicious or malicious activities such as unusual packet patterns or potential attacks.

It helps in identifying threats and understanding how intrusion detection systems work in real-world networks.

⚙️ How it works

Continuously monitors incoming and outgoing network packets

Analyzes traffic patterns

Flags abnormal or suspicious behavior

🎯 Purpose

Understand intrusion detection mechanisms

Learn basic network defense techniques

Gain practical cybersecurity experience
