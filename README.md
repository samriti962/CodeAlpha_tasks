# CodeAlpha_tasks
This repository includes cybersecurity projects developed during the CodeAlpha Internship. These projects are focused on learning network traffic analysis, packet inspection, and intrusion detection through practical implementation.

🕵️‍♂️ Project 1: Basic Network Sniffer
🔍 What this project does

The Basic Network Sniffer captures live network packets and shows key details such as:

Source IP address
Destination IP address
Protocol used

It helps in understanding how data travels across a network and how packets move between different devices.

⚙️ How it works

Monitors network traffic in real time
Captures packets using Python libraries
Extracts and displays packet data for analysis

🎯 Purpose

Learn the basics of packet sniffing
Understand network communication
Build a strong foundation in cybersecurity and networking

🚨 Project 2: Network Intrusion Detection System (NIDS)
🔍 What this project does

The Network Intrusion Detection System (NIDS) observes network traffic and identifies suspicious or harmful activities like unusual packet behavior or possible attacks.

It helps in detecting threats and understanding how intrusion detection systems function in real-world scenarios.

⚙️ How it works

Continuously tracks incoming and outgoing network traffic
Analyzes patterns in the data
Detects and flags abnormal or suspicious activities

🎯 Purpose

Learn how intrusion detection systems operate
Understand basic network security practices
Gain hands-on experience in cybersecurity
